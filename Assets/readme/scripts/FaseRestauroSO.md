# FaseRestauroSO.cs

## Descrizione
ScriptableObject che rappresenta una singola fase nel processo di restauro di un oggetto (es. Pulizia, Assemblaggio, Incollaggio).

## Responsabilità
- Contenere le informazioni descrittive e le istruzioni destinate al giocatore per la fase corrente.

## Proprietà Chiave
- `descrizioneFase` (`string`): Il testo di spiegazione o istruzioni che viene mostrato nella UI del workbench di restauro durante questa fase.
